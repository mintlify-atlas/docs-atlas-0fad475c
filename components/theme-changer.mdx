ThemeChanger is an advanced theme customization component that allows admins to modify colors, fonts, and design tokens in real-time, with changes broadcast to all connected clients via Socket.IO.

## Broadcasting mechanism

When an admin changes the theme, it broadcasts to all clients:

```typescript Theme broadcasting (admin only)
// Admin changes theme
const broadcastTheme = () => {
  if (socket && isAdmin) {
    const theme = {
      '--color-primary': primaryColor,
      '--color-surface': surfaceColor,
      '--color-bg': bgColor,
      '--color-secondary': secondaryColor,
      '--color-danger': dangerColor,
      '--color-plots': plotsColor,
      '--font-family': fontFamily,
    };
    socket.emit('broadcast_theme', theme);
  }
};

// All clients receive and apply
socket.on('theme_update', (theme) => {
  applyTheme(theme);
});
```

<Warning>
  Only admin users can broadcast theme changes. Team dashboards and public views receive updates but cannot initiate broadcasts.
</Warning>

## Persistence

Theme changes are stored in localStorage:

```typescript
localStorage.setItem('auction_theme', JSON.stringify(theme));
```

On page load, saved theme is automatically applied.

## Design notes

- **Modal interface** - Opens in full-screen modal for focused editing
- **Real-time preview** - See changes immediately before broadcasting
- **Reset option** - Return to default theme with one click
- **Admin-only broadcast** - Prevents chaos from multiple theme sources
