NeoBadge is a small, bold status indicator component designed for displaying labels, statuses, and tags with high visual impact.

## Props

<ParamField path="variant" type="string" default="neutral">
  Visual style variant:
  - `success` - Green background for positive states
  - `danger` - Red background for critical/error states
  - `warning` - Surface color for warnings
  - `info` - Secondary blue for informational badges
  - `neutral` - Default background for standard labels
</ParamField>

<ParamField path="children" type="React.ReactNode" required>
  Badge content (text or elements)
</ParamField>

<ParamField path="className" type="string">
  Additional CSS classes for custom styling
</ParamField>

## Usage examples

### Auction status indicators

From the admin panel header:

```tsx Admin status display
<NeoBadge
  variant={
    isRunning
      ? "success"
      : isPaused
        ? "neutral"
        : isCompleted
          ? "danger"
          : "neutral"
  }
>
  {status.replace("_", " ")}
</NeoBadge>
```

### Round indicators

```tsx
<NeoBadge variant="info">
  {getRoundLabel(currentRound)}
</NeoBadge>
```

### Plot status badges

From the admin plot table:

```tsx
<NeoBadge
  variant={
    row.original.status === "sold"
      ? "success"
      : row.original.status === "unsold"
        ? "danger"
        : row.original.status === "active"
          ? "info"
          : "neutral"
  }
>
  {row.original.status}
</NeoBadge>
```

### Team connection status

```tsx
<NeoBadge variant={isConnected ? "success" : "danger"}>
  {isConnected ? "Live" : "Offline"}
</NeoBadge>
```

## Styling

NeoBadge uses CSS variables for theming:

- `--color-success` - Success variant background
- `--color-danger` - Danger variant background
- `--color-surface` - Warning variant background
- `--color-secondary` - Info variant background
- `--color-bg` - Neutral variant background
- `--color-border` - Border color (2px)
- `--neo-radius` - Border radius

## Design notes

- **Uppercase text** - All badge text is automatically uppercased
- **Bold weight** - Uses font-bold for maximum readability
- **Compact size** - Small padding (px-2 py-1) for inline use
- **Border emphasis** - 2px border matches neo-brutalist style
- **High contrast** - Color choices ensure text is always readable
