## What is AuctionFest?

AuctionFest (also known as Planomics Auction) is a real-time, multi-team auction platform designed for urban planning games and educational simulations. Teams compete to acquire plots of land across a virtual city through strategic bidding across six dynamic rounds.

## Key features

### Real-time Socket.IO connection

All teams connect through persistent WebSocket connections for instant bid updates, plot adjustments, and live leaderboard changes:

```typescript
// from context/socket-context.tsx:44
const socketUrl =
  process.env.NEXT_PUBLIC_BACKEND_URL ||
  `http://${window.location.hostname || "localhost"}:8000`;

globalSocket = io(socketUrl, {
  transports: ["polling", "websocket"],
  upgrade: true,
  autoConnect: true,
  reconnection: true,
  reconnectionAttempts: Infinity,
});
```

### Team authentication

Teams log in using their team name and passcode. The system stores the team ID in localStorage for session persistence:

```typescript
// from app/page.tsx:58
localStorage.setItem("team_id", team.id);
router.push("/dashboard");
```

### Interactive plot visualization

Each plot is rendered as an animated SVG polygon with dynamic highlighting based on ownership and status:

```typescript
// from app/dashboard/page.tsx:70
<motion.polygon
  points={svgData.points}
  fill="rgba(250, 204, 21, 0.5)"
  stroke="#facc15"
  strokeWidth="4"
  initial={{ pathLength: 0, fillOpacity: 0 }}
  animate={{ pathLength: 1, fillOpacity: 0.5 }}
  transition={{
    pathLength: { duration: 1.5, ease: "easeInOut" },
    fillOpacity: { duration: 1, delay: 0.8 },
  }}
/>
```

## The auction flow

<Steps>
  <Step title="Round 1 — Initial bidding">
    Teams bid on plots with starting prices. Admin controls plot presentation and timing.
  </Step>
  <Step title="Rounds 2 & 3 — Policy adjustments">
    Admin presents policy questions that affect plot values. Teams see adjustments reflected in real-time.
  </Step>
  <Step title="Round 4 — Final bidding + rebid market">
    Teams can sell owned plots at markup (1x–1.1x value) or bid on new/resold plots.
  </Step>
  <Step title="Rounds 5 & 6 — Final adjustments">
    Last policy changes before final scores are calculated.
  </Step>
  <Step title="Final leaderboard">
    Score = Net Worth + (₹10,00,00,000 × plots owned)
  </Step>
</Steps>
